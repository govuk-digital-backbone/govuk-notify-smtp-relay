# SMTP Relay -> GOV.UK Notify

This basic Python application is designed to relay emails as an SMTP server to the GOV.UK Notify service. It listens for incoming emails, processes them to Markdown, and forwards them to Notify.

> [!IMPORTANT]  
> This relay is an internal, unofficial tool provided as-is. It is currently not supported or endorsed by the GOV.UK Notify team and Digital Backbone doesn't guarantee it'll be maintained. Evalutate and use at your own risk.
> Digital Backbone does however welcome pull requests and suggestions for improvements.
